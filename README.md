# pcbwaydnsstuff.py
well, my mailserver completely blocks a lot of ips based on their assigned country.
nevertheless, being able to receive some emails from pcbway.com daunted necessary.
wrote a script gets through all those spf records, includes and whatnot and then addst this to an ipset. that ipset is included in my firewall-rules as being ok to reach out to port 25.
